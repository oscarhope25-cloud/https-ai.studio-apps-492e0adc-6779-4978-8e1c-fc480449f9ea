Blackshadow Library - WordPress Integration

The Root Cause of "useState of null" Errors:

WordPress environments (especially when using plugins like Gutenberg, Elementor, or block-based themes) pre-load their own customized, global versions of React on the page. When raw HTML code blocks attempt to inject external CDN links for React (unpkg.com/react), the two React versions collide on the global window context. This corrupts the internal hook dispatchers, throwing the useState of null error and crashing to a black screen.
